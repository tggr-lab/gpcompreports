    About GPCompReports

    GPCompReports is a database of active-inactive residue-residue contact changes across {{ total_gpcrs }} human Class A G protein-coupled receptors. For each receptor, active and inactive AlphaFold multistate models (del Alamo et al., 2022) were compared using Residue-Residue Contact Scores (RRCS), producing a ΔRRCS value for each detected contact pair.

    Each receptor report maps contact-score changes to transmembrane segments, GPCRdb generic positions, residue-level summaries, gnomAD missense variants, AlphaMissense predictions, and conservation annotations. The database is designed for both receptor-level inspection and cross-GPCR comparison of recurrent activation-associated contact changes.

    Positions and contact pairs highlighted by GPCompReports should be interpreted as model-derived structural candidates, not direct experimental evidence of function.
