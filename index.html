    About GPCompReports

    GPCompReports provides a comparative analysis of residue-residue contacts across all 283 human Class A G protein-coupled receptors. Using AlphaFold multistate-predicted structures in both active and inactive conformations (del Alamo et al., 2022), we computed Residue-Residue Contact Scores (RRCS) to identify conformational changes associated with GPCR activation.

    Each receptor's contact network was analyzed to identify significant conformational changes, map these changes to transmembrane domains and conserved positions (GPCRdb Ballesteros-Weinstein numbering), and correlate them with human genetic variant data from gnomAD and pathogenicity predictions from AlphaMissense.

    The database enables cross-GPCR comparison, identification of Core Functional Residues (CFRs) conserved across the receptor superfamily, and exploration of structure-function-variant relationships across all 283 human Class A GPCRs.
