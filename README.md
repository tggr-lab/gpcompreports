# GPCompReports

Comparative residue-residue contact analysis of 283 human Class A GPCRs
across active and inactive conformations. A project of the
[TGGR Laboratory](https://github.com/tggr-lab) at Tel Aviv Sourasky Medical Center.

**Beta site:** https://tggr-lab.github.io/gpcompreports/
